# 🎮 Rock, Paper, Scissors Game (Python)

A **console-based Rock, Paper, Scissors game** written in **Python**.  
This fun game allows two players (or a player vs computer) to compete by choosing between rock, paper, and scissors.

---

## 🕹️ Game Rules

Rock, Paper, Scissors is a classic hand game:

- ✊ Rock beats ✌ Scissors  
- ✌ Scissors beats ✋ Paper  
- ✋ Paper beats ✊ Rock  
- Same choice = Draw

The first to score more wins after several rounds is the winner.
